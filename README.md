## 시스템 아키텍쳐

- `Web Layer`는 `Usecase Layer`에 의존하고 있다.

- `Usecase Layer`는 `Domain Layer`에 의존하고 있다.

- `Data Layer`는 `Domain Layer`을 의존하고 있으며, Repository의 구현체들이 위치하고 있다.

- `Domain Layer` 내부에서는 의존하는 상황이 발생하지만, 다른 Layer는 절대로 의존하지 않는다.
