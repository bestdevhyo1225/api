### 시스템 아키텍쳐

- `Web`은 `Usecase`에 의존하고 있다.

- `Usecase`는 `Domain`을 의존하고 있다.

- `Domain`은 어떤 Layer도 의존하지 않는다.

- `Data`는 `Domain`을 의존하고 있으며, Repository의 구현체들이 위치하고 있다.
